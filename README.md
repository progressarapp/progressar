# 💪 PROGRESSAR

App de progressão de carga inteligente baseado no **Método Paulo Muzy**.

## 🚀 Acessar o app

👉 **[Abrir PROGRESSAR](https://SEU-USUARIO.github.io/SEU-REPO/)**

## 📲 Instalar como app (PWA)

1. Abra o link acima no Chrome ou Safari
2. Toque em **"Adicionar à tela inicial"**
3. Use como app nativo — funciona offline!

## ✨ Funcionalidades

- Progressão automática de carga (Método Muzy)
- Cadência de treino com metrônomo e beep
- Histórico e gráficos de evolução
- Planta do Progresso + Modo Aposta
- Modo Iniciante e Coach virtual
- 100% offline — tudo salvo localmente
